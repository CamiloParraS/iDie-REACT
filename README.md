# iDie Frontend

Aplicacion web de gestion clinica para pacientes, desarrollada con React + TypeScript + Vite.

## Deploy

![Deploy](/deploy.jpeg "deploy")

- **Disponibilidad:** El backend utiliza un plan gratuito que suspende el servicio por inactividad. La primera petición puede tardar unos segundos más de lo habitual mientras el servidor "despierta".
- **Persistencia:** La base de datos es volatil. Los datos creados se eliminan automáticamente cuando el servidor entra en estado de reposo o se reinicia.

## Configuracion

1. Instala dependencias:

```bash
npm install
```

```bash
npm run dev      # entorno de desarrollo
npm run build
npm run preview
npm run lint
```
